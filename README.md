# Truncate Vue

Vuejs to truncate string

## Install:


 `npm install truncate-vue --save`
 `yarn add truncate-vue`

  ``` js
  import TruncateVue from 'vue-truncate-filter'
  Vue.use(TruncateVue)
  ```

## Usage:

### Vue 2.x
 ```html
  {{ text | truncate(50) }}
 ```
## Params:
- length (Number)
  
- clamp (String)
